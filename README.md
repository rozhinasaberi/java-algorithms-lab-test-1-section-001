# rojinasaberi_COMP254_001LabTest1

A COMP254 lab test project with a Streamlit tutor app that explains the core algorithm and linked-list exercises interactively.

## What This Project Is About

This repository contains a Java lab test with short problems on finding the largest values in an array, checking palindromes recursively, and building a singly linked list. It now also includes a Streamlit tutor app that turns each exercise into a guided learning activity.

This is a **Java algorithms lab test project with an AI tutor style Streamlit app**.

## What The Tutor App Teaches

- how to reason about top-k style array problems
- how palindrome recursion works from the outside in
- how singly linked lists connect nodes one step at a time
- how to explain a coding answer clearly, not just produce it

## Project Structure

- `app.py` - Streamlit tutor app
- `requirements.txt` - Python dependencies for the tutor app
- `src/com/exercise1/rojina/saberi/`
- `src/com/exercise2/rojina/saberi/`
- `src/com/exercise3/rojina/saberi/`
- `src/module-info.java`

## Run The Streamlit App

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Open The Java Project

Import into Eclipse and run the exercise classes from the `src/` folder.
